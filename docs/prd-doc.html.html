二、背景与目标

业务背景

云机汇为雀巢特批奶粉部门提供数字化营销服务，目前已交付汇直通（门店办单小程序）、能量加油站（消费者扫码营销小程序）、雀巢NHS后台（服务端管理后台）三大系统。渠道方面，除汇直通外还包括小河马、智导等第三方售卖公司。

三峡（雀巢健康科学会员中心）是另一套为雀巢服务的C端消费者营销小程序，负责会员码展示与扫码集罐兑换，三峡有独立的后台服务端。

本次"雀巢益生菌买赠活动"项目旨在对现有买赠活动机制进行升级，支持后台管理员创建和管理大单礼活动，门店店员通过汇直通小程序完成会员资格核验与产品扫码出库办单，消费者通过能量加油站小程序扫码参与兑奖。

迭代目标与成功指标

  • 后台管理员可自主创建、编辑、启用/禁停大单礼活动
  • 门店店员可通过汇直通完成完整的"查活动→核会员→扫产品→办单"流程
  • 消费者可通过能量加油站查看活动、扫码兑奖
  • 后台管理员可查询办单记录、管理活动门店

3.2 核心使用场景

场景一：后台管理员创建大单礼活动

作为雀巢运营人员，我想要在NHS后台创建一个大单礼活动（设置名称、时间、图片、规则、关联产品），以便门店店员可以在汇直通中看到该活动并为消费者办单。

验收条件：

  • 可填写活动名称（≤50字，不含空格）、选择活动类型、设置起止时间
  • 可上传活动封面图、列表图、消费者广告图（各有限制尺寸与格式）
  • 可填写广告图跳转地址（小程序路径）
  • 支持富文本编辑活动规则
  • 保存后活动出现在活动列表中

场景二：门店店员为消费者办单

作为门店店员，我想要在汇直通中选择活动、核验消费者会员资格、扫描产品罐底码完成办单，以便消费者可以获得活动参与资格并后续兑奖。

验收条件：

  • 可查看本门店可参与的活动列表（区分进行中/已结束）
  • 可选择产品和规格，查看对应礼品库存
  • 可扫描消费者三峡会员码核验资格（有资格→确认办单 / 无资格→提示原因）
  • 可扫描产品罐底溯源码出库（支持删除重扫、校验码是否已被使用）
  • 办单成功后展示订单信息，支持继续办单或查看活动数据

场景三：消费者扫码查看活动

作为消费者，我想要在能量加油站扫描产品码查看积分和活动信息，以便了解可参与的大单礼活动。

验收条件：

  • 能量站首页展示扫码入口和健康营地等内容
  • 扫码成功后展示获得积分和活动广告入口
  • 点击广告可进入大单礼活动落地页 ⚠️ 待开发
